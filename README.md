The RGB Game!
--------------

TECH STACK:
1. HTML
2. CSS
3. Vanilla JS
4. NodeJS
5. Express

DESCRIPTION:
1. You just need to guess the correct RGB Color.
2. There are two difficulties 

      --> Easy (You have to guess the color out of given 3 colors).
      
      --> Hard (You have to guess the color out of 6 given colors).
